
# AI 実践的データ入力スキル評価ツール (AI-Powered Data Entry Skill Trainer)

<img src="https://storage.googleapis.com/aistudio-programmatic-assets/project_b03bd5ea27c040d1a49f7e52b2f63e63/app_icon.png" alt="App Icon" width="150">

AIが生成する実践的な課題を通じて、データ入力の正確性とスピードを測定・向上させるためのWebアプリケーションです。ユーザーは様々な業務シナリオと難易度を選択し、リアルタイムでのタイマー計測と、AIによる詳細なフィードバックを受けることができます。

This is a web application designed to measure and improve data entry accuracy and speed through practical exercises generated by AI. Users can choose from various business scenarios and difficulty levels, and receive real-time timer tracking and detailed feedback from an AI coach.

## ✨ 主な機能 (Key Features)

- **🤖 AIによる問題自動生成:** Google Gemini APIを利用し、実践的なデータ入力問題をリアルタイムで生成します。
- **📋 多様な業務シナリオ:** 「顧客名簿」「商品リスト」「売上データ」「請求書情報」など、実際の業務に近いシナリオを選択可能。
- **📊 3段階の難易度設定:** 「簡単」「普通」「難しい」から、自分のスキルレベルに合った問題に挑戦できます。
- **⏱️ タイマー機能:** 入力時間を正確に計測し、スピードを意識したトレーニングが可能です。
- **✅ 即時採点と詳細な結果表示:** 正答率、入力時間、ミスした箇所を即座にフィードバックします。
- **💡 AIによるパーソナルコーチング:**
    - **パフォーマンスフィードバック:** 全体的なパフォーマンスに対する励ましと具体的な改善アドバイスを提供します。
    - **ケアレスミス分析:** ミスの傾向をAIが分析し、具体的な原因と対策を提案します。
- **🖥️ 実践的なUI:** 2画面表示とシートの最大化機能を備え、実際の業務に近い環境で練習できます。
- **ガイド機能:** 初めてのユーザーでも迷わず使えるように、丁寧な「ご利用ガイド」を用意しています。

## 🛠️ 使用技術 (Technology Stack)

- **Frontend:** React, TypeScript, Tailwind CSS
- **AI Model:** Google Gemini API (`@google/genai`)
- **Deployment:** Vercel

## 🚀 利用方法 (How to Use)

1.  **問題の選択:** トップページで「業務の種類」と「難易度」を選択し、`[問題を作成]`ボタンをクリックします。
2.  **入力開始:** 問題が生成されたら、`[開始 (Start)]`ボタンをクリックしてタイマーを開始し、左の「入力元データ」を見ながら右の「入力用シート」にデータを入力します。
3.  **採点:** すべての入力が終わったら、`[採点する (Submit)]`ボタンをクリックします。
4.  **結果の確認:** 採点結果、AIからのフィードバック、ケアレスミスの分析などを確認し、次の練習に活かしましょう。
5.  **新しい問題へ:** `[別の問題に挑戦する]`ボタンで、トップページに戻ります。

## 💻 ローカル環境での実行方法 (Running Locally)

### 前提条件
- Node.js (v18 or later)
- npm, yarn, or pnpm

### 手順
1.  **リポジトリをクローン:**
    ```bash
    git clone https://github.com/YOUR_USERNAME/data-entry-trainer.git
    cd data-entry-trainer
    ```

2.  **依存関係をインストール:**
    ```bash
    npm install
    ```

3.  **環境変数の設定:**
    プロジェクトのルートディレクトリに `.env` ファイルを作成し、お持ちのGoogle Gemini APIキーを設定します。

    ```.env
    API_KEY=YOUR_GEMINI_API_KEY
    ```

4.  **開発サーバーを起動:**
    ```bash
    npm run dev 
    ```
    ブラウザで `http://localhost:5173` （ポートは異なる場合があります）にアクセスしてください。
